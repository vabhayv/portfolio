# 🌐 Personal Portfolio Website

## 📌 Overview  
This is my personal portfolio website built to showcase my skills, projects, certifications, and experience as a Computer Science graduate. It serves as a central hub for recruiters and collaborators to explore my work.

---

## 🚀 Features  
- Responsive and clean UI design  
- Navigation bar for smooth section access  
- About section with skills, education, and experience  
- Projects showcase with descriptions and links  
- Interactive tab system using JavaScript  

---

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** Git, GitHub  

---

## ⚙️ How It Works  
1. Users land on the homepage with a quick introduction  
2. Navigate through sections using the navbar  
3. Explore projects with descriptions and GitHub links  
4. View certifications visually  
5. Learn more about my background in the About section
